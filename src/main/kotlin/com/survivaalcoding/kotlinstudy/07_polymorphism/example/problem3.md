# 연습문제 3

문제 2에서 이용한 `A`클래스나 `B`클래스의 인스턴스를 각각 1개씩 생성하여, `List`에 차례로 담는다.

그 후에 `List`의 요소를 차례대로 꺼내 각각의 인스턴스의 `b()` 메서드를 호출 하여야 한다. 이상을 전제로 다음 물음에 답하시오.

1. `List` 변수의 타입으로 무엇을 사용하여야 하는가
    - `Y` 컴파일 시에 `b()`를 사용하려면 정의하고 있는 `Y` 클래스를 사용해야 한다. 인스턴스 생성은 `A`, `B` 클래스로 하기 때문에 각 클래스의 `b()` 메서드가 호출된다. 
2. `val list = listOf<Y>(A(), B())`  
`list.forEach { it.b() }`
