# 연습문제 2

1. `val obj: X = A()`로 `A`인스턴스를 생성한 후,
    변수 `obj`에서 호출할 수 있는 메소드를 `a()`, `b()` `c()` 중에 골라보시오.
    - `a()` 메소드
    - 타입을 `X`로 했기 때문에 컴파일 시에 `a()` 메서드만 접근 가능하다.

2. `val y1: Y = A()`, `val y2: Y = B()`로 `A`와 `B`의 인스턴스를 생성한 후
    `y1.a()`, `y2.a()`를 실행했을 때에 화면에 표시되는 내용을 말하시오.
    - `y1.a()` 실행 시 "Aa"가 출력된다.
    - `A` 타입의 인스턴스로 생성했기 때문에 실행하면 `A`의 오버라이드 된 `a()` 메서드가 실행된다.
    - `y2.a()` 실행 시 "Ba"가 출력된다.
    - `B` 타입의 인스턴스로 생성했기 때문에 실행하면 `B`의 오버라이드 된 `a()` 메서드가 실행된다.
