# 파일 조작

### 파일 쓰기

```kotlin
// 파일 지정
val file = File("save.txt")

// 내용 쓰기
file.writeText("Hello world!")
```

### 파일 읽기

```kotlin
val file = File("save.txt")
val text = file.readText()
print(text)
```

| 함수             | 설명                             |
|----------------|--------------------------------|
| readText       | 파일 내용을 모두 String으로 읽음          |
| readLines      | 줄 단위로 List<String>으로 읽음        |
| writeText      | String 전체를 파일로 기록함             |
| writeLines     | 리스트의 각 항목을 줄로 기록함              |
| bufferedReader | BufferedReader 객체 반환, 고속 처리 가능 |
| bufferedWriter | BufferedWriter 객체 반환, 고속 처리 가능 |
