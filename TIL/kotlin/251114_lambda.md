# 람다 lambda
## 1급 객체
- 변수에 대입 가능한 객체를 1급 객체(first class object, first class citizen)라고 한다.
- 대표적인 1급 객체: 값, 인스턴스, 함수

## 람다식
자바와 비슷한데, 파라미터에 마지막에 람다식이면 함수 or 메서드를 사용할 때 소괄호를 생략해서 사용할 수 있다.  
이를 후행 람다라고 한다.

```kotlin
list.sortedWith(object: Comparator<Int> {
    override fun compare(o1: Int, o2: Int): Int {
        return o1 - o2
    }
})
// 위의 내용을 아래처럼 쓸 수 있다
list.sortedWith { o1, o2 -> o1 - o2 }
```

## 고계 함수
`fillter()`, `map()`, `toList()` 등등 많이 존재하지만 외울 필요는 없고 결국엔 자주 쓰이는 것만  
사용하기 때문에 쓰다보면 자연스럽게 익히게 된다.
