# 제네릭(Generics)

Kotlin의 제네릭은 타입을 매개변수처럼 전달받아 다양한 타입을 안전하게 처리할 수 있게 해주는 기능입니다. 클래스나 함수를 정의할 때 내부에서 타입을 고정하지 않고, 사용 시점에 외부에서 타입을 지정하도록
일반화하는 방식입니다.

- 사용하는 시점에 타입을 원하는 형태로 정의할 수 있습니다.
- 타입 안전(type safe) 효과

```kotlin
class Pocket<E> {
    private var _data: E? = null

    fun put(data: E) {
        _data = data
    }

    fun get(): E? = _data
}

fun main() {
    val pocket = Pocket<Int>()
}
```

이용 가능한 타입에 제약을 추가한 Pocket 클래스

```kotlin
class Pocket<E : Book> {}
```

Advanced

```kotlin
// 읽기 전용 (Covariance(공변성))
class Pocket<out T> // Kotlin
List < ? extends T > // java

// 쓰기 전용 (Contravariance (반공변성))
class Pocket<in T> // Kotlin
List < ? super T >

// 둘 다 됨 (Invariance(무변성))
class Pocket<T> // Kotlin, Java
```
