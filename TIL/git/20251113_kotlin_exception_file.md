# Kotlin

## 오늘 배운것

개념
- 예외 처리
  - syntax error, runtime error, logic error의 3종류
  - 예외 처리를 할때는, runtime error를 대처한다.

- 파일 조작
  - 파일을 연다
  - 파일을 읽거나 쓴다
  - 파일을 닫는다

- 여러가지 데이터 형식
  - 공공데이터는 여러가지 파일 형식으로 데이터를 제공해준다.
  - csv, pdf, html, hwp, xml, json 등등으로 제공해준다.
  - 인간과 컴퓨터가 읽기 쉬운 데이터 형식은 xml보다 json형식이다.
  - 직렬화: 클래스 -> json
    - 데이터 구조나 객체 상태를 저장하고 나중에 재구성할 수 있는 포맷으로 변환하는 과정
    - 객체를 파일의 형태 등으로 저장하거나 통신하기 쉬운 포맷으로 변환하는 과정을 의믜
    - 클래스 내부의 필드에 다른 클래스가 있다면 모두 직렬화 처리를 해줘야 함
  - 역직렬화: json -> 클래스
