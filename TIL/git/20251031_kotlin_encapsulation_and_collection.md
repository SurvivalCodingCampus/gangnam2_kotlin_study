# Kotlin

## 오늘 배운것

개념
- init블록은 생성자가 호출되는 시점에 호출 되어진다.
- 생성자에 var, val이 없으면 getter, setter가 없는것으로 간주함, 즉 멤버가 아님 
- backing filed는 프로퍼티의 값을 저장하기 위한 실제 필드 값

캡슐화에 대한 생각
- 개발자의 잘못된 접근에 대한 제어 방법을 제공 -> 외부개발자가 내 코드를 잘 사용하게끔 하려는 방법
- 현실세계와 동일하게 프로그래밍하려는 제어방법

컬렉션 정리
- key, value 쌍 : Map
- 중복 가능 : List
- 중복 불가 : Set
- 순서 중요 : List
- 순서 안 중요 : Set
- 검색 속도 중요 : Set
