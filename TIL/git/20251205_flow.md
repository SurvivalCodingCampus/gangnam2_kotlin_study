### Flow
- 콜드 스트림 / 핫 스트림 개념
  - 콜드스트림: 데이터를 구독하면 데이터 스트림을 생성
  - 핫스트림: 언제나 데이터 스트림을 생성

- LiveData와 StateFlow의 차이
  - LiveData는 안드로이드 종속
  - StateFlow는 안드로이드와 별개, 그러나 발행과 구독 관리를 해줘야함

- Flow연산자
  - map, filter, collect, zip, combine 등 다양한 연산자가 존재함
